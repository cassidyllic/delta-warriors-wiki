# Accueil

## Introduction

**Delta Warriors** est un jeu de combat 3D avec des éléments de monde ouvert et de gacha basé sur **Deltarune**. Il a été développé par Idyll Works et est inspiré par **Like a Dragon** (anciennement Yakuza).
